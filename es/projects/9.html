MEDISCANN

Role

Year

Aquí hay algunos de los proyectos en los que he trabajado

A comprehensive system composed of a React Native mobile client (Expo), a React web dashboard for administration, and a Python-based AI backend. The core features a convolutional neural network (MobileNetV2) exposed via a REST API (Flask/FastAPI) to classify dermatological conditions from user images in real-time. The entire architecture is containerised and orchestrated using Docker Compose.
